# Blockchain Attendance App

## Description
The Blockchain Attendance App is a Flutter-based application developed as an **Initial Software Product (Minimum Viable Product)** for the Mission Capstone course.

The application demonstrates a clear user interface, structured navigation, and a blockchain-ready attendance management system. While this version does not yet deploy a live blockchain or MetaMask integration, it implements the **architecture, data models, and service layers** required for secure, immutable attendance storage on the blockchain in future versions.

This MVP focuses on:
- Frontend development
- Environment setup
- Navigation and layout
- Blockchain and wallet integration planning

## Blockchain & Wallet Integration (MVP Scope)

This initial version implements a **blockchain-ready architecture**.

- Attendance data is modeled using structured Dart models.
- Wallet and blockchain interactions are handled through a dedicated service layer.
- Blockchain storage is **simulated** to demonstrate how attendance records will be stored immutably.

### MetaMask & Wallet (Planned Integration)
In future versions, the application will integrate:
- MetaMask wallet connection
- Smart contracts for attendance storage
- Ethereum test networks (e.g., Sepolia)
- Real transaction hash verification

For this MVP, wallet and blockchain logic are intentionally abstracted to:
- Reduce deployment risk
- Allow feedback during early development
- Maintain a working, demonstrable application

This approach aligns with the **Initial Software Product** requirements.
